# SymptoCheck

SymptoCheck is a simple open-source web app that lets users select symptoms and get possible causes for their condition.

## Features

- 100% Offline
- No tracking, ads, or analytics
- Lightweight and mobile-friendly
- Built with HTML, CSS, and JavaScript

## How to use

1. Open the `index.html` file in any browser
2. Select your symptoms
3. Get suggestions instantly
